"What should I expect?"

NodeSchool is a self-directed learning environment, where you bring your own laptop to learn. Attendees work on the Node workshops from nodeschool.io, but if you have a personal project you are working on, do it! The goal of NodeSchool is to help people explore and learn Node. Mentors can help you when you hit any road blocks.

"What is a workshopper?"

Workshoppers are the open source lesson modules used by NodeSchool. All are self guided (you don't need to join us in person to do one) and most work offline.

"How do I run a workshopper?"

You need to install and then run the workshoppers you want. For example, to set up and run the Beginner Node Workshopper, open a terminal and type:

$ npm install -g learnyounode
$ learnyounode

Having trouble installing Node? No worries - we'll do our best to help you get set up once you arrive at a workshop.

Motivation

Where do I find someone who can help my team? How do I discover a project worth contributing to? Introducing NodeSchool theams! This listing is an attempt to keep ephemeral projects active and shared amongst the web dev apprentices, the mentors, but also the non-tech business co-founders wishing to stay plugged into the tech community.

Besides being a list we've wanted to have, it's a fantastic (or so we think) way of using sheetsee.js. Sheetsee.js helps you hook a Google Spreadsheet (awesome for data and collaboration!) to a website to make data more visual (maps, tables, charts). Fork the Hack Spots repo and use in your own projects.

Theamed?

As learning vehicles, projects die quickly. A team, however, advances their collective skill across multiple small projects. Members stick together, longer. A team who cares about a theme over a series of projects — a themed team — is a theam. Do not underestimate a theams' change-the-world potential!
